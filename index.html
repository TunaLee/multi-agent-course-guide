워크숍 사전 설치 가이드

복사 → 붙여넣기 → 엔터, 위에서부터 차례대로만 하면 됩니다. 먼저 본인 운영체제를 고르세요.

⚠️ 준비물 — Claude Code는 Claude 유료 플랜(Pro 이상), Codex 앱은 ChatGPT 유료 플랜(Plus 이상)이 필요해요. 각 명령은 오른쪽 복사 버튼으로 그대로 복사할 수 있어요.
기본 도구
1

Homebrew

맥용 설치 도구예요. ⌘ + Space → “터미널” 을 열고 아래를 붙여넣으세요.

zsh — 터미널
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
설치가 끝나면 화면에 뜨는 “Next steps”의 두 줄(eval …)을 복사해 한 번 더 실행하세요.
2

Git · Node · Bun

개발 필수 도구 3종을 설치해요.

zsh — 터미널
$ brew install git node$ brew install oven-sh/bun/bun
에이전트
3

Claude Code CLI

터미널에서 돌아가는 코딩 에이전트예요.

zsh — 터미널
$ curl -fsSL https://claude.ai/install.sh | bash
새 터미널을 열고 claude 를 입력하면 브라우저 로그인 창이 떠요.
4

Codex 앱

⌥OpenAI의 코딩 에이전트 앱이에요. 아래에서 받아 설치하세요.
↓ openai.com/codex — macOS 앱 다운로드
  1. 다운로드한 파일 열기 → 설치
  2. Codex 앱 실행 → ChatGPT 계정으로 로그인
앱이 열리고 로그인되면 완료. (CLI 설치는 필요 없어요 — 앱에 다 들어 있어요.)
설치 확인
5

제대로 깔렸는지 확인

zsh — 터미널
$ claude --version$ git --version && node --version && bun --version
각각 버전 숫자가 나오면 성공. Codex는 앱이 로그인돼 있으면 OK.
플러그인
6

Superpowers

Claude Code

claude 를 실행한 다음, 그 안에서 아래를 차례로 입력하세요.

claude
> /plugin marketplace add obra/superpowers-marketplace> /plugin install superpowers@superpowers-marketplace
/exit 로 끄고 claude 를 다시 켜면 적용돼요.
Codex

Codex 앱에서 Plugins 사이드바 → Coding → Superpowers → + (또는 /plugins → superpowers 검색 → Install) 후 재시작.

gstack

Claude Code
zsh — 터미널
$ git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git ~/.claude/skills/gstack$ cd ~/.claude/skills/gstack && ./setup

Codex
zsh — 터미널
$ git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git ~/.codex/skills/gstack$ cd ~/.codex/skills/gstack && ./setup --host codex
Codex 앱을 재시작하면 스킬이 잡혀요.
관찰
8

lazyagent

에이전트들이 무슨 일을 하는지 트리로 보여주는 관찰 도구예요.

zsh — 설치
$ brew tap chojs23/homebrew-tap$ brew install --cask lazyagent
zsh — 훅 등록 · 실행
$ lazyagent init claude$ lazyagent init codex$ lazyagent
lazyagent 실행 후 패널 3 (Agents/subagents) 에서 서브에이전트 트리를 볼 수 있어요.
