lazyagent

에이전트들이 무슨 일을 하는지 트리로 보여주는 관찰 도구예요.

Claude Code CLI

앞 단계(Git·Node)에서 깐 npm으로 설치해요. PowerShell 또는 명령 프롬프트에서:

Windows PowerShell / cmd
> npm install -g @anthropic-ai/claude-code
claude --version 로 설치 확인. 인증은 다음 단계(API 키)에서. 구독이 있으면 claude 입력 → 브라우저 로그인으로 대체돼요.
⚠ 잘 안 될 때 — Windows 자주 겪는 2가지

① claude 인식 안 됨 (CommandNotFoundException · PATH) — 터미널을 닫고 새 창으로(PATH는 새 세션부터 적용). 그래도면 npm config get prefix 경로를 사용자 PATH에 추가.

② claude.ps1 … 실행할 수 없으므로 (UnauthorizedAccess · 위 화면) — PowerShell 실행 정책이 .ps1을 차단. 현재 사용자만 허용으로 바꿔요:

Windows PowerShell
PS> Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

→ Y 입력. 회사 정책(Group Policy)으로 덮여 안 바뀌면: 관리자 PowerShell에서 -Scope LocalMachine으로 실행하거나, 명령 프롬프트(cmd)에서 claude 실행(=claude.cmd, 실행 정책 영향 없음).

lazyagent

에이전트들이 무슨 일을 하는지 트리로 보여주는 관찰 도구예요. Git Bash 또는 WSL에서 실행해요. 설치는 ~/.local/bin에 되는데 PATH에 없으면 lazyagent: command not found가 나요 — 아래 첫 두 줄로 PATH에 추가하면 해결돼요.

Git Bash / WSL — 훅 등록 · 실행
$ echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc$ source ~/.bashrc$ lazyagent init claude$ lazyagent init codex$ lazyagent
