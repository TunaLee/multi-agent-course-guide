워크숍 사전 설치 가이드

복사 → 붙여넣기 → 엔터, 위에서부터 차례대로만 하면 됩니다. 먼저 본인 운영체제를 고르세요.

⚠️ 준비물 — Claude Code는 Anthropic API 크레딧(또는 Claude 유료 구독), Codex 앱은 ChatGPT 유료 플랜(Plus 이상)이 필요해요. 각 명령은 오른쪽 복사 버튼으로 그대로 복사할 수 있어요.
기본 도구
1

Homebrew

맥용 설치 도구예요. ⌘ + Space → “터미널” 을 열고 아래를 붙여넣으세요.

zsh — 터미널
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
설치가 끝나면 화면에 뜨는 “Next steps”의 두 줄(eval …)을 복사해 한 번 더 실행하세요.
2

Git · Node · Bun

개발 필수 도구 3종을 설치해요.

zsh — 터미널
$ brew install git node$ brew install oven-sh/bun/bun
에이전트
3

Claude Code CLI

터미널에서 돌아가는 코딩 에이전트예요.

zsh — 터미널
$ curl -fsSL https://claude.ai/install.sh | bash
설치 완료 — 인증은 다음 단계(API 키)에서 합니다. 이미 Claude 유료 구독이 있으면 claude 실행 → 브라우저 로그인으로 대체할 수 있어요.
4

API 키 등록 ANTHROPIC_API_KEY

Claude Code가 API 크레딧으로 동작하도록, 제공된 키를 환경변수로 등록해요.

🔑워크숍에서 API 키를 제공해요 — 아래 목록에서 본인 키를 받으세요(직접 만들 필요 없음). 키는 sk-ant-api03-… 로 시작하고 비밀번호처럼 취급.
↗ 워크숍 API 키 목록 (Notion)

받은 키를 환경변수로 등록 (zsh):

zsh — 환경변수 등록
$ echo 'export ANTHROPIC_API_KEY="sk-ant-api03-여기에-받은키"' >> ~/.zshrc$ source ~/.zshrc
echo $ANTHROPIC_API_KEY 로 키가 보이면 등록 완료. 이후 claude 실행 시 “이 API 키를 쓸까요?”에 승인하고, /status 로 인증 방식을 확인하세요.
5

Codex 앱

⌥OpenAI의 코딩 에이전트 앱이에요. 아래에서 받아 설치하세요.
↓ openai.com/codex — macOS 앱 다운로드
  1. 다운로드한 파일 열기 → 설치
  2. Codex 앱 실행 → ChatGPT 계정으로 로그인
앱이 열리고 로그인되면 완료. (CLI 설치는 필요 없어요 — 앱에 다 들어 있어요.)
설치 확인

zsh — 터미널
$ claude --version$ git --version && node --version && bun --version
각각 버전 숫자가 나오면 성공. Codex는 앱이 로그인돼 있으면 OK.

lazyagent

에이전트들이 무슨 일을 하는지 트리로 보여주는 관찰 도구예요.

zsh — 설치
$ brew tap chojs23/homebrew-tap$ brew install --cask lazyagent
zsh — 훅 등록 · 실행
$ lazyagent init claude$ lazyagent init codex$ lazyagent

🔑워크숍에서 API 키를 제공해요 — 아래 목록에서 본인 키를 받으세요(직접 만들 필요 없음). 키는 sk-ant-api03-… 로 시작하고 비밀번호처럼 취급.
